【LeetCode】不同路径II

63. 不同路径 II - 力扣（Leetcode）

题目

思路

试了试bfs和dfs，都超时了

由于这个问题具有无后效性和最优子结构这两个特性，故要用动规来解题

由于只能向右或者向下走，所以状态转移方程为：
$$
dp[i][j] =
\begin{cases}
dp[i-1][j] + dp[i][j-1], & obstacleGrid[i][j] = 0 \
0, & obstacleGrid[i][j] = 1
\end{cases}
$$
边界条件：
$$
dp[i][0] = 1 (obstacleGrid[i][0] = 0, 0 <= i < m)\
dp[0][1] = 1 (obstacleGrid[0][i] = 0, 0 <= i < n)
$$

上述代码时间复杂度为O(mn)，空间复杂度为O(mn)

因为dp[i][j]只与dp[i-1][j]和dp[i][j-1]有关，因此我们可以利用滚动数组方式求解

每一轮的dp[j]都是继承自上一轮的
