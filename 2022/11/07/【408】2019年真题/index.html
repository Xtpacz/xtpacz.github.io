t11 最佳归并树虚段的个数

假如这个树是一个最佳归并树（ K路归并 ）。那么假设叶子节点（初始归并段）个数为 N0，则有这个式子成立：(N0 - 1) mod (K - 1) = 0
现在已知叶子结点是120个，12路归并，问要添加几个虚段（ 也就是还少几个叶结点？）
假设少 x 个，则根据上面那个等式，则下面这个等式也要成立：
(120 + x - 1) mod (12 - 1) = 0
解出来 x = 1（ x取满足条件的最小正整数，取多了没意义 ）

t22 DMA流程

DMA传送前 由CPU 执行几条IO指令（用来测试IO设备状态），向DMA控制器的有关寄存器置初值、设置传送方向、启动该设备等

脑袋里不要只死记硬背是CPU呀，
操作系统得依靠叫做 设备驱动程序 的东西来和硬件打交道

“驱动程序一般指的是设备驱动程序（Device Driver），是一种可以使计算机和设备进行相互通信的特殊程序。相当于硬件的接口，操作系统只有通过这个接口，才能控制硬件设备的工作，假如某设备的驱动程序未能正确安装，便不能正常工作。因此，驱动程序被比作“ 硬件的灵魂”、“硬件的主宰”、“硬件和系统之间的桥梁”等。

t44
簇(块)的大小是由文件系统（软件）来决定de
而扇区的大小是由硬件决定的，（物理格式化划分柱面、磁道和扇区~）

1、物理(低级)格式化：划分柱面、磁道和扇区，and对已经损坏了的磁道和扇区做“坏”标记
2、分区：对磁盘的整体空间做划分（C盘、D盘。。。），这些分区可以安装不同OS和软件、存储文件。。
3、逻辑(高级)格式化：对扇区进行逻辑编号、简历逻辑盘的引导记录、文件分配表、文件目录表和数据区

扇区：实际读写的基本单位
簇，操作系统分配存储空间基本单位（比如，若簇大小是1KB，OS存一个1B的文件，那么就要给它分配1KB，虽然有些浪费。。）

因为不同硬件的内部功能结构都不完全一样，所以硬件厂家都会提供自己的驱动程序，来让大家用的操作系统可以通过这个驱动程序来和他家的硬件打交道。
所以将簇号转换成物理地址的过程，由磁盘驱动程序（设备驱动程序）完成。
